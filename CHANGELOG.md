# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-18

### Added
- Initial release
- Multi-dimensional complexity analysis (cyclomatic, architectural, algorithmic)
- Support for OpenAI GPT-4 and Anthropic Claude providers
- GitHub repository integration via API
- Rich CLI interface with progress indicators
- Comprehensive test suite with >80% coverage
- Docker support with multi-stage builds
- CI/CD pipeline with GitHub Actions
- Automated linting, testing, and security scanning
- Token management and cost tracking
- JSON export functionality
- Type-safe Pydantic models
- Detailed documentation and examples

### Security
- Non-root Docker container execution
- Environment-based API key management
- Automated security scanning in CI pipeline

## [Unreleased]
